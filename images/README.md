# Productfoto's

Sla hier de officiële Kennametal productfoto's op.
De app laadt ze automatisch — als een foto ontbreekt, wordt de SVG-tekening getoond.

- `kmt-kentip-hpg.jpg`  — HPG (staal)

- `kmt-kentip-hpl.jpg`  — HPL (RVS)
- `kmt-kentip-feg.jpg`  — FEG (vlakke bodem)
- `kmt-kentip-spf.jpg`  — SPF (CFRP/composiet)

## Foto's downloaden
1. Ga naar kennametal.com
2. Zoek het inserttype op (bijv. "KenTIP FS GTP")
3. Sla de productfoto op als jpg/png met bovenstaande bestandsnaam
4. Upload naar deze map in de GitHub repo

Aanbevolen formaat: ~400×400 px, witte achtergrond.
